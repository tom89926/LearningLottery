## Lucky Draw Feature

This is a simple lucky draw feature where you can input some names and then draw pairs.

### Frontend with UI Framework

If your frontend uses a UI framework, you can follow these steps to use the feature:

1. Download or clone this project to your local environment.
2. Navigate to the directory with frontend using UI framework.
3. Run `npm install` in the command line to install dependencies.
4. Run `quasar dev` to start the frontend server.
5. Open `http://localhost:5173` in your browser to view the interface.
6. Input names and press the button to see the results.

### Frontend without UI Framework

If your frontend does not use a UI framework, you can follow these steps to use the feature:

1. Download or clone this project to your local environment.
2. Navigate to the directory with frontend without UI framework.
3. Run `npm install` in the command line to install dependencies.
4. Run `npm run dev` to start the frontend server.
5. Open `http://localhost:5173` in your browser to view the interface.
6. Input names and press the button to see the results.

### Backend

The backend is implemented using .NET 7 Web API. You can follow these steps to start the backend service:

1. Download or clone this project to your local environment.
2. Navigate to the backend directory.
3. Run `dotnet build` in the command line to build the project.
4. Run `dotnet run --launch-profile "https"` to start the backend service with HTTPS protocol.
5. The backend service will run on `https://localhost:7061` and `http://localhost:5026`.

### Notes

- Make sure your browser is compatible with the technologies used in this project.
- Please follow the appropriate input format, and ensure that the entered names can be processed correctly.
